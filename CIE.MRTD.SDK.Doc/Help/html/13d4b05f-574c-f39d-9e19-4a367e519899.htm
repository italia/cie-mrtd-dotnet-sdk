ByteArrayRemoveBT1 Method Carta d'Identità Elettronica - MRTD SDK
Ritorna un array di bytes ottenuto rimuovendo il padding BT1 dal parametro data. Il padding BT1 è costituito dalla sequenza di bytes {0x00,0x01,0xff,...,0xff,0x00} anteposta all'array iniziale Solleva un'eccezione se il padding non è corretto
