SmartCard ClassCarta d'Identità Elettronica - MRTD SDK
L'oggetto SmatCard permette di effettuare operazioni sui lettori di smart card, monitorare l'inserimento e la rimozione di una smart card, aprire un connessione e inviare comandi.
