EACTerminalAuthentication Method Carta d'Identità Elettronica - MRTD SDK
Effettua il protocollo di Terminal Authentication. Viene ricostruita la catena di certificati dalla CVCA presente sull'MRTD fino al certificato di inspection system contenuto in IS usando i certificati contenuti nella lista chain. I dati da firmare vengono elaborati tramite il binding specificato e la firma viene effettuata tramite il delegate signFunc.
